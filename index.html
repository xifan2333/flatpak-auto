Reusable Flatpak repository for personal projects This page is generated from product metadata and updated automatically by GitHub Actions whenever a tracked upstream release changes.

Add the remote

Run this once on a machine. After that, every package below can be installed by app ID.

flatpak remote-add --if-not-exists xifan https://xifan2333.github.io/flatpak-auto/xifan.flatpakrepo
